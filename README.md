# My Portfolio

This is the source code for my personal portfolio website, built with React, TypeScript, and Vite.

**[View the live portfolio](https://peres.github.io/my-portfolio)**

## Built With

-   **Frontend:**
    -   [React](https://reactjs.org/)
    -   [TypeScript](https://www.typescriptlang.org/)
    -   [Vite](https://vitejs.dev/)
    -   [Tailwind CSS](https://tailwindcss.com/)
-   **Deployment:**
    -   [GitHub Pages](https://pages.github.com/)

## Installation

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/peres/my-portfolio.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd my-portfolio
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm run dev
    ```
